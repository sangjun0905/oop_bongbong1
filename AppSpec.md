1) 한 프롬프트당 **한 줄**, **값만** 출력. 설명/기호/따옴표/마크다운/콜론/접두사 금지. 앞뒤 공백 금지. **마지막에 줄바꿈 1개**.\

상태 → 허용 형식(정규식):\
- 메인 메뉴(화면에 1~4와 '>'가 보임): `^[1-4]$`\

- 검색 키워드: Name `^[A-Za-z]{1,15}$` / StudentID `^\d{1,10}$` / Admission `^\d{4}$` / Birth `^\d{4}$` / Department `^[A-Za-z]{1,23}$`\
